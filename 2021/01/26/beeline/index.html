1. c3s 생성 (네이버 하둡 클러스터)

여태 docker pull -> run & 컨테이너명 변경 -> start
-> exec -it {container_name} /bin/bash 이런식으로 접속했었고
컨테이너 안에서 shell을 빠져나온다고 해서 컨테이너가 종료되지는 않았었다.

현재는 docker start & docker attach 하는데
해당 shell 빠져나오면 컨테이너가 종료됨.
